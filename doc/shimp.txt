                                             *:ShimpToggle*
Jump to source/header.

:ShimpToggle {arg}
    Open window normally, same as :e[dit] {file}: >
        :ShimpToggle open
<    Other options are:                            >
        :ShimpToggle left
        :ShimpToggle right
        :ShimpToggle top
        :ShimpToggle below
<    Sample keymaps:                              >
        nnoremap <silent> <leader>she :ShimpToggle open<cr>
        nnoremap <silent> <leader>shV :ShimpToggle left<cr>
        nnoremap <silent> <leader>shv :ShimpToggle right<cr>
        nnoremap <silent> <leader>shS :ShimpToggle top<cr>
        nnoremap <silent> <leader>shs :ShimpToggle below<cr>
<
                                             *shimp_options*
    Default options:                             >
        g:shimp_options = {
            headers: ['h', 'hpp'],
            sources: ['c', 'cpp', 'cc'],
            header_dirs: ['.', 'include'],
            source_dirs: ['.', 'src'],
        }
<    In order to change these, you can manually modify the
    dict or put something in *after*.
